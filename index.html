A Microsoft começou a desenvolver o Microsoft Windows em setembro de 1981. Os primeiros Windows, como o 1.0, 2.0, são compatíveis apenas com partições formatadas em sistema de ficheiros FAT, ou como é chamado, FAT 16. O 3.x poderia ser instalado em FAT 32, porém necessita ser instalado o MS-DOS 7.10, que era incluído nos disquetes de inicialização do Windows 95 OSR2 e Windows 98, necessitando modificar alguns arquivos para permitir seu funcionamento. Ao mudar do 3.1 para o 95B (Windows 95 OSR 2/OSR 2.1), os HD's poderiam ser formatados em FAT 32. Inicialmente lançado com o Windows NT, a tecnologia NTFS é agora o padrão de fato para esta classe. Com a convergência de ambos sistemas, o Windows XP passou também a preferir este formato, até hoje é o formato utilizado nos Windows. Atualmente o Windows é o sistema operacional mais usado do mundo.

Logo após o lançamento do Windows 1.0 em novembro de 1983 a Microsoft buscou cada vez mais evoluir seu Sistema Operacional para cada dia deixa-lo melhor tanto para quem o utiliza quanto para a empresa em si, a partir disso diversas atualizações foram sendo lançadas ao longo dos anos, chegando cada vez mais perto de um sistema perfeito tendo bons exemplos disso como o Windows 7,10 e XP, porém nem sempre dando tão certo, sendo assim o caso do Windows Vista.

Bill Gates é um empresário americano mundialmente conhecido por fundar a Microsoft, a mais conhecida empresa de software do mundo. É também reconhecidamente um dos pioneiros na revolução do computador voltado para o consumidor comum. Desde a década de 1990, Bill Gates é figura constante entre as pessoas mais ricas do mundo, ocupando por diversas vezes a primeira posição.
